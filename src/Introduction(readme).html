Technology Point

This tiny project is mainly developed in the environment of Ubuntu 14.04. And the key point used in this project is the teconology of CSS3, HTML5 and some JS. The animation in the pages are achieved by CSS3 and the canvas is based on Flower Power, a charming project in HTML5. JS is used to control the whole process. So, there is nothing difficult.
